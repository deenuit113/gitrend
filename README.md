> 웹 접근성 이해도 높이기 위한 토이프로젝트

## 주요 기능
> Github API를 이용해 Github 내 인기 있는 주제, 저장소 모음
> 다크모드
> 메모장

---

## 웹 접근성을 위해
- 단축키를 사용하여 모든 기능에 대해 키보드 탐색이 가능하도록 구현
- 버튼 내의 텍스트 TTS 기능 (Web Speech API)
- 버튼 내의 텍스트 확대 기능
- 부분적 시각약자를 위한 강조하는 버튼 UI 구성
- 메모장 음성인식 STT 기능 (Web Speech API)
- 글자와 배경색 명암비 4.5 이상 (큰 글자의 경우 3.0) - __WCAG 1.4.3 통과 기준__
- Aria Label 사용
- 다크모드

---

## 고려사항 & 추후 개발 방향
- 단축키 도움말

- 명료한 HTML 구조
